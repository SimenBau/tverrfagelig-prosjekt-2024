Det første jeg lærte i programmering er hvordan innstalere Visual studio code. Vi skulle også innstalere mange utvidelser for å gjøre visual studio code bedre å bruke.
I programering har jeg lært å bruke HTML, CSS og JS. Det første skole-halvåret brukte vi på å lære og skrive HTML og CSS, jeg syntes dette var det mest interesante. Det andre skole-halvåret brukte vi på JS, jeg syntes JS er mye vanskeligere og jeg liker ikke JS.
Noe av det mest morsome jeg har lært i programering i år er Flexbox. Flexbox bruker man når man skal endre posisjonen og rekkefølgen på ting. For å lære oss Flexbox, brukte vi siden Flexbox Froggy.
